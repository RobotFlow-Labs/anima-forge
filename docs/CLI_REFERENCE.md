one JSON document on stderr and exit nonzero. Progress output is suppressed in JSON mode.

## `forge pipeline`

```text
Usage: forge pipeline [OPTIONS]

  Run the FORGE distillation pipeline end-to-end.

Options:
  --config TEXT                   Config file path  [default:
                                  configs/forge_nano.yaml]
  --stage TEXT                    Single stage:
                                  labels|distill|compress|export|validate
  --skip-labels / --no-skip-labels
                                  Skip teacher label generation  [default: no-
                                  skip-labels]
  --device TEXT                   Device: auto|cuda|cpu|mps
  --max-steps INTEGER             Override max distillation steps
  --batch-size INTEGER RANGE      Override distillation batch size  [x>=1]
  --gradient-accumulation-steps INTEGER RANGE
                                  Override distillation gradient accumulation
                                  [x>=1]
  --max-episodes INTEGER RANGE    Limit teacher-label episodes for bounded real-
                                  data runs  [x>=1]
  --output-dir PATH               Override paths.output_dir from the config
  --data-dir PATH                 Override paths.data_dir from the config
  --checkpoint FILE               Trained checkpoint to use for compress,
                                  export, or validate
  --allow-mock                    Explicitly permit mock inputs (all artifacts
                                  are stamped MOCK)
  --help                          Show this message and exit.
```
